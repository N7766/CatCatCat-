# 猫咪钓鱼点击游戏 🐱🎣

一个可爱又功能丰富的增量点击游戏，与软乎乎的猫咪一起钓鱼，收集鱼鱼，升级装备，解锁成就！

![游戏预览](https://img.shields.io/badge/游戏-猫咪钓鱼点击游戏-pink)

## ✨ 游戏特色

- 🐱 **软乎乎的猫咪主角** - 点击猫咪时会挤压变形，非常治愈
- 🎣 **多种升级系统** - 强化鱼竿、自动钓鱼、幸运加成、猫猫伙伴等
- ⭐ **转生机制** - 转生获得海星，永久提升收益
- 🏆 **成就系统** - 完成各种里程碑成就，获得奖励
- 📖 **鱼鱼图鉴** - 收集各种鱼类，解锁图鉴奖励
- 🎨 **外观定制** - 自定义猫咪颜色、鱼竿样式、鱼图标和背景主题
- 💥 **暴击系统** - 包含超暴击特效，带来视觉冲击
- 🌈 **环境事件** - 极光、流星等特殊事件，提供临时增益
- 😊 **猫咪表情系统** - 猫咪会根据游戏状态变化表情（兴奋、困倦等）
- 💾 **自动保存** - 游戏进度自动保存到本地存储

## 🎮 游戏玩法

### 基础操作

1. **点击猫咪** - 点击屏幕中央的猫咪即可开始钓鱼
2. **升级装备** - 在右侧商店购买升级，提升收益
3. **自动收益** - 购买自动钓鱼助手后，即使不点击也能获得鱼鱼
4. **转生系统** - 积累一定数量的鱼后，可以转生获得海星，永久提升收益

### 升级系统

- **强化鱼竿** 🎣 - 提升每次点击的基础收益
- **自动钓鱼助手** 🤖 - 每秒自动获得鱼鱼
- **幸运小鱼干** 🍀 - 增加暴击概率和暴击伤害倍率
- **猫猫伙伴** 🐱 - 提供全局收益倍率加成

### 转生系统

- 达到 10,000 条鱼后可以转生
- 转生会重置所有鱼和升级
- 转生后获得**海星**货币（永久保留）
- 每颗海星永久增加 +5% 所有收益
- 可以在海星商店购买永久升级物品

### 海星商店

- **深海罗盘** - 每级额外 +5% 全局收益
- **幸运星项链** - 每级额外 +2% 暴击率
- 所有海星物品在转生后保留

### 成就系统

完成各种里程碑目标，包括：
- 总鱼数里程碑（100, 1K, 10K, 100K, 1M...）
- 每秒收益里程碑（10, 50, 100, 250, 500, 1000...）
- 各种游戏内成就

### 鱼鱼图鉴

- 收集不同类型的鱼类
- 完成图鉴里程碑获得奖励
- 每条鱼都有独特的收集次数里程碑

### 外观定制

- **猫咪颜色** - 多种可爱的猫咪毛色可选
- **鱼竿样式** - 原木、金色、竹制、水晶、宇宙等多种鱼竿
- **鱼图标** - 更换主界面的鱼图标样式
- **背景主题** - 白天/夜晚主题，或根据时间自动切换

### 特殊事件

- **金色鱼事件** - 随机出现的特殊事件，点击可获得大量鱼鱼
- **环境事件** - 极光、流星等，提供临时增益效果
- **超暴击** - 高倍率暴击触发全屏特效

## 🚀 快速开始

### 本地运行

1. 克隆或下载项目
2. 直接用浏览器打开 `index.html` 即可开始游戏
3. 无需安装任何依赖或运行服务器

## 🎯 游戏目标

1. **短期目标** - 积累鱼鱼，购买升级，提升收益
2. **中期目标** - 完成成就，解锁外观，收集图鉴
3. **长期目标** - 多次转生，积累海星，购买永久升级，追求更高收益

## 💡 游戏技巧

1. **早期阶段** - 优先升级"强化鱼竿"，提升点击收益
2. **中期阶段** - 投资"自动钓鱼助手"，实现被动收益
3. **后期阶段** - 升级"幸运小鱼干"提升暴击，投资"猫猫伙伴"获得倍率加成
4. **转生时机** - 当升级成本过高时考虑转生，获得海星购买永久升级
5. **海星投资** - 优先购买"深海罗盘"，全局收益加成最划算

## 🔧 技术实现

- **纯前端实现** - 无需后端，所有数据保存在浏览器本地存储
- **响应式设计** - 适配各种屏幕尺寸
- **动画效果** - CSS3 动画和 JavaScript 动画结合
- **SVG 图标** - 所有图标使用 SVG 实现，支持自定义颜色
- **模块化代码** - 代码结构清晰，易于维护和扩展
